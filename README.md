## Запуск проекта

Чтобы запустить проект локально нужно выполнить след действия.

1. Создать в корне .env файл и вставить туда содержимое из env.example(файл лежи в корне проекта)
2. SMTP_USER и SMTP_PASSWORD напсать свою действоющую почту и чтобы был включен протокол для smtp
3. HTTP_FRONT, HTTP_FRONT_CONFIRM, HTTP_FRONT_REPLACEPASSWORD указать ссылки на фронт
4. После чего выполнить след команду

```bash
$ docker-compose up
```
## След. дейсвтия для запуска 
5. Перейти на http://localhost:9000/ и ввести логин и пароль с env файла.
6. В облаке нужно сгенерировать привытный и публичный ключ и вставить его MINIO_ACCESS_KEY и MINIO_SECRET_KEY
7. После чего пересобрать проект 


## Последующие запуски нужно делать только через  
```bash
$ docker-compose up
```

## PS

Эти команды выполнять не надо
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
